## Overview :sparkles:

## Development :computer:
### Requirements
- Node.js, NPM, pNpM

### Setup
- Clone repo
```sh
$ git clone https://github.com/aidantrabs/portfolio-v2
```
- Install dependencies 
```sh
$ pnpm i
```
- Launch local server
```sh
$ pnpm dev
```
